# PeriPrint (perripage-ferrum)

Десктопное GUI-приложение для печати на термопринтерах Peripage (A6/A6+/A40/A40+) через Bluetooth SPP/RFCOMM.

Даёт удобный UX поверх низкоуровневой библиотеки [`peripage`](https://pypi.org/project/peripage/) (bitrate16/peripage-python): drag & drop файла → предпросмотр в реальном чёрно-белом растре → печать в 1-2 клика, с сохранённым профилем принтера.

**Статус:** черновик / RFC — реализация ещё не начата.

## Проблема

Печать на Peripage сейчас — это CLI-команды с ручным подбором MAC-адреса, модели принтера, ширины и нарезки картинки на части из-за перегрева печатающей головки. PeriPrint убирает это трение.

## Возможности (MVP)

- Bluetooth-скан и сохранение профилей принтеров (имя, MAC, модель).
- Drag & drop файлов: PNG/JPG/BMP, TXT, PDF (постранично).
- Растеризация под ширину конкретной модели принтера + дизеринг для превью 1:1 с физической печатью.
- Автонарезка длинных изображений на чанки с паузами — защита от перегрева.
- Очередь печати с прогрессом по чанкам, реконнект и резюм после обрыва Bluetooth-соединения.
- История печати.

Полный список требований (P0/P1/P2), архитектура, модель данных и план разработки — в [`periprint-spec.md`](periprint-spec.md).

## Стек

Python 3.11+, `customtkinter`, `tkinterdnd2`, `peripage`, `Pillow`, `PyMuPDF`.

Целевая платформа — Linux (тестируется на Linux Mint/Cinnamon); Windows/macOS — best effort, не блокируют MVP.
